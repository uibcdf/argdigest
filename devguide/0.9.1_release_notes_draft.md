# ArgDigest 0.9.1 Release Notes (Draft)

## Release Position

`0.9.1` is a stabilization patch in the `0.9.x` pre-1.0 consolidation window.
No new features are introduced; this release tightens evidence, docs, and
integration confidence.

## What is included

- New MolSysMT-like representative integration tests:
  - package-style digestion with cross-argument injection,
  - argument-name standardization pattern (`name -> <element>_name`).
  - file: `tests/test_molsysmt_representative_paths.py`.
- Showcase coverage expanded with explicit decorator-style integration page:
  - `docs/content/showcase/decorator-style-integration.md`.
- Devguide synchronization updates for:
  - `0.9.x` consolidation tracking,
  - `1.0.0` checklist evidence references,
  - `1.0.0` release/migration summary draft.

## Validation snapshot

- `pytest -q`: green.
- `make -C docs html`: green (existing warning baseline unchanged).

## Compatibility notes

- Public API unchanged.
- No planned breaking changes before `1.0.0`.
- This patch is fully compatible with current `0.9.0` integration paths.
